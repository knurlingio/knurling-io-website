API

v.0.0.1 of the API returns results from legacy Canadian weightlifting competitions in JSON.


NOTE: This API is experimental and highly volatile. Please contact knurlingio@gmail.com if you want to use it. Knurling.IO reserves the right to modify, restrict access and/or terminate the API at its discretion.


The API is intended to support development of embeddedable data visualizations, such as the following (built with React and D3.js):



  • Available Endpoints:
    url: http://api.knurling.io/
    This will return all endpoints supported by the API.
  • Athletes (all):
    url: http://api.knurling.io/athletes
    This will return all athletes available to the API. Filtering will be implemented soon.
  • Athletes (single):
    url: http://api.knurling.io/athletes/{athlete_id}
    This will return an athlete's details as well as any results they've earned in competition.
  • Athletes' Events:
    url: http://api.knurling.io/athletes/{athlete_id}/events
    This will return a list of events in which the athlete participated.
  • Events (all):
    url: http://api.knurling.io/events
    This will return a list of all events available to the API. Filtering will be implemented soon.
  • Events (single):
    url: http://api.knurling.io/events/{event_id}
    This will return details of a single event including any results associated with that event.
