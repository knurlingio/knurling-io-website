Adipower in CSS


CSS for digital art? I've seen some great experiments recreating company logos using only CSS.

The general consensus is that they are cool but somewhat impractical; they wouldn't be used in any production capacity.

I decided to try my hand at recreating a lifting shoe in CSS - after hours spent tweaking values in terminal on one screen, watching the browser in the other - I'm inclined to agree: this wasn't very practical.

With that said, I learned more about CSS3 and SCSS during this little project than any other time, and that's something I can take into other work.

Maybe I'll try to beat my time on Romaleos next time.
